=====================================================
Directory Structure
=====================================================

.. include:: ../../includes_repository/includes_repository_directory.rst

The sub-directories in the |chef| repository are:

.. list-table::
   :widths: 150 450
   :header-rows: 1

   * - Method
     - Description
   * - :doc:`essentials_repository_structure_certificates`
     - Stores SSL certificates that are generated by the ``rake ssl_cert`` command.
   * - :doc:`essentials_repository_structure_chef`
     - A hidden directory that stores .pem files and the |knife rb| file.
   * - :doc:`essentials_repository_structure_config`
     - Contains the |rake| configuration file: ``rake.rb``.
   * - :doc:`essentials_repository_structure_cookbooks`
     - Contains cookbooks that have been downloaded from the |url cookbook| or created locally.
   * - :doc:`essentials_repository_structure_data_bags`
     - Stores data bags (and data bag items) in |json| (.json).
   * - :doc:`essentials_repository_structure_environments`
     - Stores environment in |ruby| (.rb) or |json| (.json).
   * - :doc:`essentials_repository_structure_roles`
     - Stores roles in |ruby| (.rb) or |json| (.json).
