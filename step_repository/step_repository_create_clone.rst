.. This is an included how-to. 


The |chef repo| is available on |github|: |url opscode_chef_repo|.

To clone the |chef repo| from |github|:

1. Download and install |git|.
2. Run the following command:

   .. code-block:: bash

      $ git clone git://github.com/opscode/chef-repo.git

3. (Optional) After the |chef repo| is cloned, the history of that repository can be wiped out by removing the ".git" directory, which allows the initialization of a new repository or to move the |chef repo| into another version source control system, such as |svn|, |mercurial|, or |bazaar|.
