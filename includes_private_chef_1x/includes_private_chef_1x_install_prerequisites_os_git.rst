|chef private| requires that |git| be installed, so that various internal services can confirm their own revision.

Install |git| on |redhat| and |centos| 6:

.. code-block:: bash

   $ yum install git

Install |git| on |ubuntu|

.. code-block:: bash

   $ apt-get install git-core
