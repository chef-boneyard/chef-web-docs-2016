This lightweight resource provider has the following actions:

.. list-table::
   :widths: 200 300
   :header-rows: 1

   * - Action
     - Description
   * - ``:create``
     - Use to create a |cloudkick| check.
   * - ``:disable``
     - Use to disable a |cloudkick| check.
   * - ``:enable``
     - Use to enable a |cloudkick| check.
