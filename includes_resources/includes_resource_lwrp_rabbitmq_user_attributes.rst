This lightweight resource provider has the following attributes:

.. list-table::
   :widths: 200 300
   :header-rows: 1

   * - Attribute
     - Description
   * - ``:password``
     - The password for the user.
   * - ``:permissions``
     - One (or more) permissions that will be set.
   * - ``:user``
     - The name of the user.
   * - ``:vhost``
     - The name of the virtual host.
