This lightweight resource provider has the following attributes:

.. list-table::
   :widths: 200 300
   :header-rows: 1

   * - Attribute
     - Description
   * - ``:name``
     - The name of a formula repository located on |github|.
   * - ``:tapped``
     - Indicates that a formula repository is tapped.
