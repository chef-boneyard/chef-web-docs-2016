The |lwrp transmission torrent file| lightweight resource is used to download a file using the |bittorrent| protocol. This lightweight resource is similar to the ``file`` and ``remote_file`` lightweight resources, but allows very fast downloads for files that are part of large |bittorrent| swarms.
