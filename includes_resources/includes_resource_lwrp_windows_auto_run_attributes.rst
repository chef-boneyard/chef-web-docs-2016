This lightweight resource provider has the following attributes:

.. list-table::
   :widths: 200 300
   :header-rows: 1

   * - Attribute
     - Description
   * - ``:args``
     - The arguments for a program.
   * - ``:name``
     - The name of a value to be stored in the |windows| registry.
   * - ``:program``
     - The program to be run.
