This lightweight resource provider has the following actions:

.. list-table::
   :widths: 200 300
   :header-rows: 1

   * - Action
     - Description
   * - ``:disable``
     - Use to disable a group.
   * - ``:enable``
     - Use to enable a group.
   * - ``:reload``
     - Use to reload a group.
   * - ``:restart``
     - Use to restart a group.
   * - ``:start``
     - Use to start a group.
   * - ``:stop``
     - Use to stop a group.
